# greenbook
*This software retrieves weather information from the Weather API*

## Installation
* Runs in Python 2.7
* Run commands: 
  * sudo pip install matlab
  * sudo pip install pandas
  * sudo pip install requests
  * sudo pip install 
  * sudo pip install sqlalchemy
  * sudo pip install
